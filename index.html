Information

This application provides a visual representation ofthe Ford-Fulkerson algorithm,which can be used to determine the maximum flow of a directed network graph. A graph can be built on the canvas above, using the instructions provided below.

On each edge is two numbers: the remaining capacity (orange) and the flow (blue).

When the algorithm is run, augmenting paths chosen to increase the flow are highlighted in green at a set interval. Once the maximum flow has been determined, all edges that have flow are highlighted in blue. The information panel to the right of the canvas will display the calculated maximum flow, the number of augmenting paths used, and a trace of each augmenting path.

Instructions

Vertices

  • Add a vertex: Left- or right-click on canvas
  • Delete a vertex: Right-click on vertex and select 'Remove vertex'
  • Move a vertex: Shift + left-click on vertex and drag to desired position
  • Set graph source: Right-click on vertex and select 'Set as source'
  • Set graph sink: Right-click on vertex and select 'Set as sink'

Edges

  • Add an edge: Left-click and drag between source and target vertices
  • Select an edge: Left click on the edge
  • De-select an edge: Hit 'Enter' or 'Esc'
  • Change capacity of an edge: Select the edge and enter a numeric value
  • Delete an edge: Select the edge and hit 'd' or 'delete'

Maximum flow

  • To run the animation:
    1. Select method of searching for augmenting paths
      • Breadth First Search
      • Depth First Search
      • Fattest Path
    2. Click 'Get max flow'
  • Use the 'Interval' number input to change the time interval between the display of augmenting paths
  • To reset the flow to zero: Click 'Reset flow'
  • To erase the entire graph: Click 'Erase'
    • Note: This cannot be undone!
